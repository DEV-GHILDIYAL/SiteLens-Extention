🚀 Feature Documentation

🔍 Contrast Analyzer

Real-time WCAG 2.1 auditing for text and images. Instantly identifies color pairs that fail accessibility standards across the entire page.

⚡ Manual Checker

Premium tool for ad-hoc color testing. Allows designers to input Hex codes and see live luminosity-aware previews with pass/fail ratios.

🔗 Link Detector

Advanced crawler with auto-scrolling logic to capture lazy-loaded links. It verifies destination titles and warns about broken paths.

🖼️ Image Audit

Scans the DOM for images missing essential 'alt' attributes or featuring duplicate sources, ensuring high screen-reader compatibility.

📝 Font Audit

Analyzes heading hierarchies (H1-H6) and font properties to ensure a clear, logical content structure for all users.

🔘 Button Audit & Analyze

Dual-mode tool to verify button casing, color compliance, and destination validity (Titles vs. targets).

🎨 Color Extractor & Theme Gen

Extracts the full brand palette from any site and generates WCAG-compliant color variations for design consistency.

📄 Content Hub (Checker & Splitter)

Verifies that specific document segments exist on-page and splits complex text into logical heading/content blocks for easier auditing.

🚫 Lorem & Placeholder Detector

Scans for common placeholder text like 'Lorem Ipsum', helping you polish your site before production deployments.

🤖 AI Self-Audit

Simulates human-like auditing to verify dynamic elements and complex UI flows that static scans might miss.
